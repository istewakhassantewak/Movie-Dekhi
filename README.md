# MovieDekhi

Static frontend portfolio-style Netflix clone landing page.

## Run Locally

- Open `index.html` directly in your browser, or
- Run `npm start` to serve the project statically.

## Deploy

- This project is now frontend-only (HTML/CSS/JS).
- No backend, database, or environment variables required.
- Works on Vercel, Netlify, GitHub Pages, and any static host.

## Vercel Deployment

- Import repo into Vercel.
- Framework preset: `Other`.
- Keep default build settings.
- Deploy.
