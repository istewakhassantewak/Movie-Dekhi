🎬 MovieDekhi – Netflix Style Landing Page

A static frontend movie streaming UI clone inspired by Netflix. Built using HTML, CSS, and JavaScript, this project focuses on modern UI design, responsive layout, and smooth user experience.

🚀 Live Demo

🔗 https://movie-dekhi.vercel.app/

📌 Features
🎥 Netflix-style UI design
📱 Fully responsive layout (mobile, tablet, desktop)
🧭 Clean navigation and structured sections
🖼️ Movie banners and category rows
🎨 Dark-themed modern interface
⚡ Lightweight static frontend project
🌐 Easy deployment on Vercel / Netlify / GitHub Pages
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
